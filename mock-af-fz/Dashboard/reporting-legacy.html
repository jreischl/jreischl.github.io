Reporting includes several great features that will improve the reporting process; queuing of reports, aggregate reporting for multiple club owners and an improved Excel export. Additionally, all reports will be kept in Generated Reports for quick reference at a later date, but can be deleted at any time.

Select Clubs

The first step in generating a report is to select club(s) from the right. The list will display only the clubs assigned to your account in Staff Management. For reports that require a date range (usage reports), select the appropriate start and end date using the date filters on the right as well.

Queue Reports

Once the club filters and dates have been selected, click the queue report button to generate that report. If there are several reports you want to generate using the same club and date filters, you can click Queue Report for each one. You do not have to wait for the first report to finish before starting a new one. The reports will then process in the order you queued them.

View Reports

When the report is ready to view, just click the view report button next to that report in the Generated Reports dropdown. You can export the report to Excel or print the report as a PDF by using the corresponding icons in the upper right hand corner.

When viewing a specific report, it is incredibly easy to sort the information in the order you wish to view it. In each column header are two sort icons that look like up and down arrows. Click the appropriate one to sort by that column whether ascending (up arrow) or descending (down arrow).

Member color counts, as well as other critical stats, including member usage, guest usage, and tanning usage, are now included in the Club At A Glance page.
